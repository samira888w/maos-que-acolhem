# Sistema de Gerenciamento de Doações

## ONG Mãos que Acolhem

Projeto desenvolvido para o Projeto Integrador Extensionista I (PIE I).

## Objetivo

Desenvolver uma aplicação web para auxiliar a ONG Mãos que Acolhem no controle de:

- Doadores
- Beneficiários
- Doações
- Entregas

## Tecnologias Definidas

- Vue 3
- Tailwind CSS
- PHP
- MySQL

## Funcionalidades do MVP

- Cadastro de doadores
- Cadastro de beneficiários
- Registro de doações
- Controle de entregas
